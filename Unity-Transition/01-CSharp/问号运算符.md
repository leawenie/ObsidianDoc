> 问号运算符的基本用法如下。
1. 三元运算
2. 可空值类型
3. 空合并

## 三元运算
> 通常用于简单的条件判断赋值。例：
```c#
int age = 18;
sting result = arg >= 18 ? "成年人" : "未成年人";
```

## 可空值类型
> 让变量可以被赋值为`null`。例：
```C#
int? age = null; //int? 等价于 Nullable<int> 
if(age.HasValue){
	Console.WriteLine(age.Value); //若变量类型被？修饰，建议通过 变量名.Value 的方式访问其值，这样会返回原始数据类型，如果直接通过变量名访问，你将得到一个string类型的。
}else{
	Console.WriteLine("值为 Null");
}
```
1. `Nullable<T>`类型的变量，在直接输出时（通过变量名直接访问或输出），实际上是调用`Nullable<T>`所重写的`ToString()`，因此直接输出或访问你将获得一个`string`类型的值。
2. 通过`变量名.Value`方法访问或输出你将获得原始类型。
3. 在性能方面，`变量名.Value`的方式更快。

## 空合并
> 通常用于防止变量被设置为`null`。例:
```C#
string? name = null;
string displayName = name ?? "未命名";

name ??= "未命名"; //若name为null，则将其赋值为"未命名"
```
